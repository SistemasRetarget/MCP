Convierte links de redes sociales en posts de WordPress usando OpenAI.

== Description ==

News Converter permite a los editores y periodistas convertir rápidamente enlaces de redes sociales (Twitter/X, artículos web) en posts de blog estructurados, usando inteligencia artificial para extraer y formatear el contenido.

**Características:**

* Extrae contenido de tweets/X automáticamente
* Análisis inteligente con OpenAI (GPT-4o)
* Interfaz de revisión y edición completa
* Genera título, extracto, autor y contenido
* Gestión de etiquetas y categorías
* Descarga automática de imágenes destacadas
* Publica como borrador o directamente

== Installation ==

1. Sube la carpeta `news-converter-plugin` a `/wp-content/plugins/`
2. Activa el plugin desde el menú 'Plugins'
3. Ve a 'News Converter > Configuración' y agrega tu API Key de OpenAI
4. Comienza a convertir enlaces desde 'News Converter'
