Tema moderno para portal de noticias con carrusel, grillas, filtros y widgets.

== Description ==

Retarget News es un tema de WordPress diseñado específicamente para portales de noticias y blogs editoriales. Incluye todas las características necesarias para presentar contenido de forma profesional y atractiva.

Características principales:
- Carrusel de noticias destacadas con auto-play
- Múltiples layouts de grilla (2, 3 y 4 columnas)
- Sidebar con widgets personalizables
- Diseño responsive (móvil, tablet, desktop)
- Sistema de breaking news
- Integración lista para plugin News Converter
- Customizer para personalizar colores
- Menú de navegación con dropdowns
- Páginas optimizadas para SEO

== Installation ==

1. Ve a Apariencia → Temas → Añadir nuevo → Subir tema
2. Selecciona el archivo retarget-news.zip
3. Instala y activa el tema
4. Configura en Personalizar (Apariencia → Personalizar)

== Setup ==

Después de activar el tema:

1. Crear menú de navegación:
   - Ve a Apariencia → Menús
   - Crea un menú y asígnalo a "Menú Principal"
   - Incluye tus categorías principales

2. Configurar widgets:
   - Ve a Apariencia → Widgets
   - Personaliza el sidebar principal
   - Configura las 3 columnas del footer

3. Personalizar colores:
   - Ve a Apariencia → Personalizar → Opciones Retarget
   - Cambia el color primario y de acento

4. Configurar Homepage:
   - Ve a Ajustes → Lectura
   - Selecciona "Tus últimas entradas"

== Demo Mode ==

El tema incluye "Demo Mode" - si no tienes el plugin News Converter instalado, mostrará datos de ejemplo para que puedas visualizar el diseño. Una vez conectado el plugin, mostrará las noticias reales de tu cola editorial.
