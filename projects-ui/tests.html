Tests + Prompt Evaluation

¿Qué es esta sección?
Aquí validamos que el sistema MCP funciona correctamente. Los tests verifican que el sistema pueda crear proyectos, procesar solicitudes, manejar errores, etc.

¿Por qué es importante?
Garantiza que los cambios que hacemos no rompan funcionalidades existentes. Es como un "check-up" automático del sistema antes de cada deploy.

¿Cómo funciona?
El MCP se auto-evalúa contra 17 casos de uso conformes a estándares internacionales. El Prompt Grader evalúa la calidad de los prompts que usamos para comunicarnos con la IA.
