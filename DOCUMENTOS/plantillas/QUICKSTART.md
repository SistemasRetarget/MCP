# ⚡ QUICKSTART - Ver el Demo en 3 Minutos

## 🥇 MÉTODO 1: Instalación Ultra-Rápida (Recomendado)

### 1. Descargar e instalar
```bash
# Si tienes WordPress local funcionando:
cd /ruta/a/wordpress/wp-content/themes/
# Copiar retarget-news-theme.zip aquí y descomprimir
unzip retarget-news-theme.zip
```

### 2. Activar en WordPress
```
→ Admin WP → Apariencia → Temas
→ Click en "Retarget News"
→ Activar
```

### 3. ¡Listo! El demo aparece solo
Abre tu sitio y verás:
- 6 noticias de demo con imágenes
- Carrusel animado
- Sidebar con widgets

---

## 🥈 MÉTODO 2: Tema Moderno (Mejor Performance)

### Requiere:
- Node.js instalado (https://nodejs.org)
- Terminal/Consola

### Pasos:
```bash
# 1. Entrar al tema
cd retarget-modern-vite

# 2. Instalar dependencias (1 minuto)
npm install

# 3. Compilar (30 segundos)
npm run build

# 4. Subir a WordPress
# Copiar toda la carpeta a /wp-content/themes/
```

### 5. Activar en WordPress (igual que método 1)

## ❓ PREGUNTAS FRECUENTES

**¿Necesito el plugin News Converter?**
→ NO. El demo funciona sin el plugin.

**¿Las imágenes son reales?**
→ Son de Unsplash (stock photos gratuitas). En producción usarás tus imágenes.

**¿Puedo cambiar las noticias del demo?**
→ Sí, creando posts reales en WordPress. El demo solo aparece si no hay posts.

**¿Funciona en móvil?**
→ Sí, es 100% responsive.
